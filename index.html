Markdown basics

Ghost Writer accepts plain text, but these common markdown patterns can help you structure your notes.

  • # Heading 1
  • ## Heading 2
  • - List item
  • 1. Numbered item
  • [Link text](https://example.com)
  • **bold** and *italic*
  • `inline code`

When you press Enter at the end of a list item, Ghost Writer continues the list for you. Numbered lists move to the next number, nested lists stay indented, and pressing Enter again on an empty list line ends the list.
